> 如果你的错误提交已经推送到自己的远程分支了，那么就需要回滚远程分支了。 

### 首先要回退本地分支：

git shell
git reset --hard 版本号

### 紧接着强制推送到远程分支：

git push -f

> 注意：本地分支回滚后，版本将落后远程分支，必须使用强制推送覆盖远程分支，否则无法推送到远程分支
